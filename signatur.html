Signatur einrichten

Apple Mail
Mail → Einstellungen → Signaturen → + → HTML-Ansicht öffnen (⌘ + Shift + H) → Inhalt einfügen

Gmail (Web)
Einstellungen → Alle Einstellungen → Signatur → Neue Signatur → Im Browser: diese Datei öffnen, Signatur-Block markieren und in Gmail einfügen

Outlook (Mac/Web)
Einstellungen → Signaturen → Neue Signatur → HTML einfügen
Hinweis: Outlook Desktop (Windows) rendert SVG nicht — in dem Fall Icons als PNG-Bilder einbetten.
